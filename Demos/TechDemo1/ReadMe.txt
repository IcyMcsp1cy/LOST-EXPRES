go to directory via command line

setup a local python environment named 'env'

enter python environment using: .\env\Scripts\activate

run app using: python app.py

build docker using: docker image build -t my-image .

run docker using: docker run -p 5001:5000 -d python-hello-world

point browser to http://localhost:5000/ or http://127.0.0.1:5000/
